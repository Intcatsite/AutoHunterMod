# AutoHunter Mod (Fabric 1.16.5)

Этот мод добавляет функцию автоматической охоты.

## Управление
*   **F** — Включить/Выключить режим автоматической охоты.

## Как это работает
При включении (ENABLED):
1.  Игрок ищет ближайшее живое существо в радиусе 20 блоков.
2.  Автоматически поворачивается к нему.
3.  Бежит в его сторону.
4.  Атакует, когда подойдет достаточно близко (с учетом кулдауна удара).
5.  После убийства цели мгновенно переключается на следующую ближайшую.

## Сборка
Для сборки мода используйте Gradle:
```bash
gradle build
```
Готовый `.jar` файл появится в папке `build/libs/`.
